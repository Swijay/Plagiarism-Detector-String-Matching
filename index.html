How This Project Works

1. Text Preprocessing

Converts text to lowercase, removes extra spaces, and prepares clean text for matching.

2. KMP Algorithm

Uses LPS array to search copied sentences efficiently without repeated comparisons.

3. Rabin–Karp

Uses rolling hash logic to compare sentence patterns with the submitted document.

4. Shingling

Creates small word groups and compares them to detect phrase-level similarity.
